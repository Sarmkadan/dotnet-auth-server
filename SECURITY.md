# Security Policy

We take the security of dotnet-auth-server seriously.

## Supported Versions

Currently, the following versions are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, we kindly ask that you responsibly disclose it to us.

Please use one of the following methods to report a vulnerability:

1. **GitHub Private Advisory**: Use GitHub's built-in private advisory feature to report the vulnerability securely.
2. **Email**: Contact us directly at rutova2@gmail.com.

We will acknowledge your report within 48 hours and work with you to address the issue promptly.
